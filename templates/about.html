About Bookworks

Bookworks is a suite of tools for converting and transforming books between different formats. Currently offering EPUB conversion with audiobook generation coming soon. Created by Rory Scott.

Key Features

  • Browser-based interface for easy file conversion
  • Command-line interface for automation and scripting
  • Multiple format conversions:
    • Markdown to EPUB with enhanced formatting
    • Markdown to Audiobook (coming soon)
  • Enhanced EPUB features:
    • Fixes broken Markdown links that span multiple lines
    • Improves paragraph spacing for better readability
    • Creates proper chapter structure based on level-2 headings
    • Adds metadata (title, author, date)
    • Optional table of contents generation
  • Debug mode for troubleshooting
  • Docker support for containerized deployment
  • Comprehensive test suite

How It Works

  1. The utility reads the input Markdown file (either through web upload or command line)
  2. Processes the content based on the selected conversion:
    • For EPUB: Formats content, fixes links, adjusts spacing
    • For Audiobook: Splits content into optimal chunks, generates audio (coming soon)
  3. Generates the output file in the selected format with proper metadata

Requirements

Local Development

  • Python 3.10 or higher
  • Poetry for dependency management
  • Pandoc installed and available in PATH

Docker

  • Docker installed on your system
  • No other dependencies needed
